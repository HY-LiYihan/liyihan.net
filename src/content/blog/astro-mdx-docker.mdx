Astro keeps the website static by default, while MDX keeps the writing format expressive. The result is a Git-managed academic site that can still embed rich interactive material.

## Why MDX

Markdown is still the right default for long-form writing. MDX becomes useful when the page needs a component, a video, or a local interactive demonstration.

## Deployment Model

The production container does not run a Node server. The Docker build compiles Astro into `dist/`, indexes the static HTML with Pagefind, and copies the final output into an Nginx image.
